接口说明

获取必应当天和前七天的每日一图

请求方式

GET

请求地址

https://lab.996workers.icu/api/bing/day
参数说明
参数 说明
day 每日一图时间，0位当前，1为昨天，2为前天，以此类推，最多为前7天
status 返回的状态码
bing 返回的每日一图信息
url 返回的每日一图地址
copyright 返回的每日一图版权
msg 返回的错误信息

请求示例

https://lab.996workers.icu/api/bing/0

返回示例

{
    "status": 1,
    "bing": {
        "url": "http://s.cn.bing.net/th?id=OHR.UhuRLP_ZH-CN5421658032_1920x1080.jpg&rf=LaDigue_1920x1080.jpg&pid=hp",

    }
}
