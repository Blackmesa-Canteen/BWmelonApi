接口说明

网易云音乐搜索、歌曲url获取、歌曲信息获取、歌词获取、歌单列表获取

请求方式

GET

请求地址

搜索：https://lab.996workers.icu/api/cloudmusic/search/key
url解析：https://lab.996workers.icu/api/cloudmusic/url/id
信息：https://lab.996workers.icu/api/cloudmusic/info/id
歌词：https://lab.996workers.icu/api/cloudmusic/lyric/id
歌单：https://lab.996workers.icu/api/cloudmusic/playlist/id
参数说明
参数 说明
key 歌曲搜索关键词
id 歌曲id或者歌单id
status 状态码
name 歌曲名或作者名
id 歌曲id
artist 作者名列表
musicurl 歌曲解析地址，格式为mp3
musiclyric 歌词
musicinfo 歌曲信息
msg 返回的错误信息
