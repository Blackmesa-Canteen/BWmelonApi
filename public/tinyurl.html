接口说明

将长网址转成短网址，支持url.cn和t.cn短网址

请求方式

GET

请求地址

https://lab.996workers.icu/api/tinyurl/urlcn/?longurl=
https://lab.996workers.icu/api/tinyurl/tcn/?longurl=
参数说明
参数 说明
longurl 需要缩短的长网址
status 返回的状态码
tinyurl 返回的短网址
msg 返回的错误信息

请求示例

https://lab.996workers.icu/api/tinyurl/urlcn/?longurl=https://lab.996workers.icu

返回示例

{
    "status": 1,
    "tinyurl": "https://url.cn/518YFDl"
}
